# Project Brief: Cantik.AI

**Created:** 2026-01-05  
**Last Updated:** 2026-03-03

## Core Identity
**Name:** Cantik.AI  
**Type:** Web Application (PWA-ready)  
**Purpose:** AI-powered skin analysis and skincare consultation platform for Indonesian users

## The Problem
Indonesian users seeking skincare advice face:
- Expensive dermatologist consultations (Rp 300k-500k per visit)
- Overwhelming product choices without guidance
- No accessible way to track skin health over time
- Trial-and-error approach wasting money

## The Solution
Free AI-powered skin analysis using camera + Google Gemini 2.0, providing:
- 7-metric skin analysis (overall health, acne, dark spots, wrinkles, hydration, texture, pores)
- Conversational AI chat for personalized advice (Groq AI)
- Analysis history tracking
- Product recommendations
- Educational content

## Core Requirements

### Must-Have Features (MVP)
- Camera-based facial analysis
- AI-powered results with insights
- User authentication (email/password)
- Analysis history
- AI chat assistant (3 modes: Fast, Thinking, Pro)
- Product catalog
- Educational articles
- Admin dashboard

### Target Users
- Primary: Urban Indonesian women, 22-35 years old
- Secondary: Budget-conscious students, 18-25 years old
- Tech-savvy, smartphone native

## Success Criteria
- MVP launch: Q1 2026
- Beta: 50 users tested
- Public launch: 1000+ users in 3 months
- Retention: 40%+ 7-day retention
- Performance: <3s page load time

## Constraints
- Budget: Bootstrap (minimal cost)
- Timeline: 2 months to MVP
- Resources: Solo developer (full-stack)
- Infrastructure: VPS hosting

## Business Model
- Phase 1: Free (user acquisition)
- Phase 2: Affiliate revenue (product recommendations)
- Phase 3: Premium tier (Rp 49k/month)
- Phase 4: B2B (white-label for clinics)

- **Phase:** Development (Phase 3)
- **Sprint:** Sprint 8 - Final Polish
- **Progress:** 95% complete
- **Next:** Testing → Deployment → Launch
